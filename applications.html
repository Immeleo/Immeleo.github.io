Applications of Cryptography

There are various applications of Cryptography including Confidentiality, Intergrity and Authentication. In this Section we focus on Digital signatures.

Digital Signatures.

Digital signatures are a means of verifying the authenticity of a document or message sent electronically just like the traditional signatures were. This is to guarantee the recipient that the document was not truly altered.

How it Works

Digital signatures rely on asymetric cryptography.Documents are hashed to create a digest. The digest is then encrypted by the sender with their private key.Digest is then embedded with the document and sent. The receiver decrypts the digest with the senders public key and calculates an hash from the document using the same algorithm used by the sender. If the calculated digest matches the decrypted digest then the receiver is assured that the document has not been altered. A digital certificate can be used to guarantee the senders identity.

Additional Security precautions

Putting the private key on a smart card. Using smart card readers with a different keyboard. Using digital signatures with only trusted applications. Using network attached security module. Use separate keys for encryption and signing
